# Movie API Project

A simple movie search application built using HTML, CSS, and JavaScript.
It uses the OMDb API to fetch and display movie information.

## Features
- Login validation
- Movie search using OMDb API
- Clean UI with CSS
- API integration using fetch

## How to Run
- Open the project using Live Server in VS Code
- Or open HTML files directly in browser for demo

## Technologies Used
- HTML
- CSS
- JavaScript
- OMDb API
